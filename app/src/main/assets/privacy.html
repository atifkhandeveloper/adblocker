I. PERMISSIONS USED BY Ad Blocker

- Precise location (GPS and network-based) & approximate location (network-based)

ACCESS_COARSE_LOCATION ACCESS_FINE_LOCATION

Ad Blocker does not recognize or save users' location information for any malicious purpose. These permissions are required to provide more accurate, localized services based on country regions, which serves the goal to improve the user experience. We will never upload any users' location information.

- Full network access & receive data from Internet

ACCESS_NETWORK_STATE INTERNET

This permission is used to access the device’s network for certain functions including downloading Virus Database and App Junk Cache Database.

- View Wi-Fi connections

ACCESS_WIFI_STATE

This permission is used in Ad Blocker settings in order to view Wi-Fi connections.

- Get, authenticate and manage accounts on the device

AUTHENTICATE_ACCOUNTS GET_ACCOUNTS MANAGE_ACCOUNTS

Ad Blocker neither recognizes nor saves users' account information for any programs. This permission is used for us to read the email when you send feedback to us so that we can contact you when problem solved.

- Get battery stats

BATTERY_STATS

This permission allows Ad Blocker to get the battery stats, in order to inform and help users to save battery.

- Directly call phone numbers & Answer phone calls

CALL_PHONE

This permission allows user directly answer or hang up the phone calls. This won’t result in unexpected fees and calls. Ad Blocker won’t upload your privacy information.

- Take pictures and videos

CAMERA

This permission is used to take photos of the intruders who want to snoop on your private apps. We will never upload any users’ pictures or videos.

- Change network connectivity

CHANGE_NETWORK_STATE

This permission is used in Ad Blocker settings and notification toolbar, in order to change network connectivity.

- Connect and disconnect from Wi-Fi

CHANGE_WIFI_STATE

This permission is used in Ad Blocker settings in order to connect and disconnect from Wi-Fi.

- Read and write App storage space

CLEAR_APP_CACHE CLEAR_APP_USER_DATA GET_PACKAGE_SIZE PACKAGE_USAGE_STATS WRITE_INTERNAL_STORAGE

Ad Blocker is allowed to detect and delete junk files on users’ phone with this permission.

- Disable keyguard

DISABLE_KEYGUARD

This permission allows Ad Blocker to provide a quick entry for features like Boost or Junk Clean on the lock screen page.

- Expand status bar

EXPAND_STATUS_BAR

This permission is only used to guide users when using notification organizer for the first time.

- Control flashlight

FLASHLIGHT

This permission is used to turn ON/OFF your device’s camera flashlight.

- Retrieve running apps

GET_TASKS

This permission is used in Boost in order to view the running apps.

- Close other Apps

KILL_BACKGROUND_PROCESSES

Ad Blocker is allowed to stop other apps and boost your phone with this permission.

- Modify audio settings

MODIFY_AUDIO_SETTINGS

This permission is used to modify the sound of notification to provide a better experience.

- Read contacts

READ_CONTACTS

Ad Blocker does not recognize or save users' contacts information for any programs. The only purpose is to help users block all annoying spam calls and provide powerful call assistance. We will never upload any users’ contacts information.

- Read the contents of your USB storage

READ_EXTERNAL_STORAGE

This permission allows users to view files stored on their SD card. We promise that we will never disclose your privacy to any person or third party interest information.

- Read phone status

READ_PHONE_STATE

The only purpose of using this permission is to identify call status so that users can directly answer or hang up the phone calls. Phone status is not read for any other purposes.

- Read sync settings

READ_SYNC_SETTINGS

This permission allows Ad Blocker to recognize whether sync settings are turned on or off.

- Receive boot completed

RECEIVE_BOOT_COMPLETED

This permission allows Ad Blocker to make the phone vibrate once after users set the device on vibrate, confirming the setting has been turned on.

- System alert window

SYSTEM_ALERT_WINDOW

This permission allows Ad Blocker to protect private applications with AppLocker. Ad Blocker won’t upload your privacy information.

- Vibration

VIBRATE

This permission allows Ad Blocker to make the phone vibrate once after users set the device on vibrate, confirming the setting has been turned on.

- Wake lock

WAKE_LOCK

This permission allows Ad Blocker to keep the phone awake during a scanning.

- Modify or delete the contents of your USB storage

WRITE_EXTERNAL_STORAGE

This permission allows users to delete files stored on their SD card. We promise that we will never disclose your privacy to any person or third party interest information.

- Toggle sync on and off

WRITE_SYNC_SETTINGS

This permission allows Ad Blocker to synchronize with the user's sync settings.

- Accessibility

BIND_ACCESSIBILITY_SERVICE

This permission is used to close apps chosen by users with accessibility. This is only used to speed up user devices and no relative information will be uploaded.

- Request install packages

REQUEST_INSTALL_PACKAGES

This permission is only used in App Manager feature to help users install apps easily. Ad Blocker won’t upload your privacy information.
